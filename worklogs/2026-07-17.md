# 2026-07-17

## 完成進度

- PivotGrid 列欄位 Header 的排序符號改為沿用 FabGrid 的升冪／降冪三角形，並依 `PivotField.descending` 顯示目前排序方向。
- 移除 PivotGrid 原本固定顯示的 `↕` 字元，補上排序符號 DOM 契約測試與 source-mode cache version。
- 修正 Grid 在展開狀態捲到最右側後縮減欄位時，scroll-linked Header 沿用舊最大捲動距離而移出畫面的問題；layout 更新後於下一個 animation frame 合併校正 Header／Footer 的最終同步距離。
- PivotGrid 疊合欄改為只在畫面隱藏而不從完整 columns 集合移除；預設 Excel 匯出保留全部疊合子欄與資料，並將這些欄位維持為 hidden column。
- PivotGrid Excel 匯出改用完整 Pivot rows；畫面疊合後未顯示的業務明細列仍寫入工作表，並依目前疊合狀態維持為 hidden row。
- PivotGrid 列欄位 Header 排序改為與 FabGrid 相同的預設／升冪／降冪三態循環；預設不顯示排序符號並保留 dimension 原始出現順序，右鍵選單同步提供清除排序。
- PivotPanel 的 Rows／Columns 欄位新增右鍵排序 popup，可直接設定預設順序、升冪或降冪；選項使用 radio active 狀態並支援 `Escape` 與點擊外部關閉。
- PivotGrid Header Cell 右鍵選單新增 Grid 全螢幕項目，沿用 FabGrid `toggleFullscreen()`、既有 icon 與 locale，並依目前狀態切換進入／離開文字。
